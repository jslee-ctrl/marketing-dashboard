# 퍼포먼스 마케팅 데이터 분석 프로젝트

## 프로젝트 개요
매일 채널 광고 데이터(channel)와 앱스플라이어 어트리뷰션 데이터(appsflyer)를 수집·조인·분석하는 퍼포먼스 마케팅 파이프라인.

---

## 폴더 구조

```
프로젝트 루트/
├── data/
│   └── raw/
│       ├── channel/          ← 채널 원본 데이터 (일별 parquet/csv)
│       └── appsflyer/        ← 앱스플라이어 원본 데이터 (일별 parquet/csv)
├── dashboard/
│   └── app.py                ← Streamlit 대시보드
└── CLAUDE.md
```

**파일 네이밍 규칙:** `YYYY-MM-DD_{source}.parquet`
- 예) `2025-01-01_channel.parquet`, `2025-01-01_appsflyer.parquet`
- parquet이 있으면 csv는 자동 스킵 (중복 방지)

## 데이터 조인 규칙

**조인 기준 키:** `채널` + `캠페인` + `그룹` + `소재`  
(날짜는 각 파일이 일별로 분리되어 있으므로 concat 후 조인)

**데이터 우선순위:**
- **전환 지표 (회원가입, 구매, 매출)** → 앱스플라이어 기준 우선 (어트리뷰션 정확도)
- **노출, 클릭, 비용** → 채널 데이터 기준 (미디어 원천 데이터)

**데이터 확정 시점:** 전날 데이터는 D+1 수집 기준. 당일 데이터는 미완성으로 간주.

## 소재 네이밍 컨벤션

**패턴:** `{포맷}_{메시지}_{시즌}_{AB버전}_{크리에이티브버전}`

```
VID_플러스멤버십_겨울_A_v1
│    │          │    │  └── 크리에이티브 버전 (v1, v2 ...)
│    │          │    └───── A/B 테스트 버전 (A=컨트롤, B=변형) ※없는 경우도 있음
│    │          └────────── 시즌 (겨울, 봄, 여름, 가을 / 또는 특정 프로모션명)
│    └───────────────────── 메시지 카테고리
└────────────────────────── 소재 포맷
```

### 소재 포맷
| 코드 | 설명 |
|------|------|
| VID | 동영상 소재 |
| IMG | 이미지 소재 |
| CRS | 카루셀(슬라이드) 소재 |
| TXT | 텍스트 소재 (검색광고) |

### 메시지 카테고리
| 코드 | 설명 |
|------|------|
| 플러스멤버십 | 유료 멤버십 가입 유도 |
| 적립혜택 | 포인트/적립 혜택 강조 |
| 할인쿠폰 | 할인 쿠폰 제공 소구 |
| 배송혜택 | 무료배송 등 배송 혜택 강조 |
| 신상품 | 신제품 출시 소구 |
| 특가 | 한정 특가 프로모션 |

## KPI 정의 및 계산식

| 지표 | 계산식 | 단위 | 설명 |
|------|--------|------|------|
| ROAS | 매출 / 비용 × 100 | % | 광고비 대비 매출. 100% = 본전 |
| CPA | 비용 / 구매 | 원 | 구매 1건당 광고비 |
| CPC | 비용 / 클릭 | 원 | 클릭 1회당 광고비 |
| CTR | 클릭 / 노출 | % | 노출 대비 클릭률 |
| CVR | 구매 / 클릭 | % | 클릭 대비 구매전환율 |
| CPR | 비용 / 회원가입 | 원 | 회원가입 1건당 광고비 |

**ROAS 표기 기준: % (퍼센트)**
- 100% = 광고비 회수 (본전)
- 300% = 광고비의 3배 매출
- 채널별 목표 ROAS 참고값:

| 채널 | 목표 ROAS | 참고 (실측 2025-01-01) |
|------|----------|----------------------|
| 네이버 (브랜드KW) | 1000% 이상 | 4,406% |
| 구글 | 500% 이상 | 1,004% |
| 메타 | 300% 이상 | 547% |

## 이상치 판단 기준

- **노출 100 미만** 소재: 통계 신뢰도 낮음, 분석 시 별도 표시
- **CPA가 채널 평균의 3배 초과**: 비효율 소재로 플래그
- **ROAS 100% 미만**: 광고비 미회수 — 즉시 점검 필요

## 분석 규칙

1. **인사이트 출력 순서:** 채널 → 캠페인 → 그룹 → 소재
2. **비교 기준:** 전주 대비 증감 포함 (데이터가 2일 이상일 때)
3. **수치 포맷:** 금액은 원화(₩), ROAS는 %, CTR/CVR은 %
4. **소재 분석 시** 포맷(VID/IMG/CRS/TXT)별로 먼저 묶어서 비교
5. **채널 분류(외부/자체) 구분** 후 해석 — 자체 채널(네이버)은 ROAS가 구조적으로 높음

## 작업 시 주의사항

- 채널 데이터와 앱스플라이어 데이터는 **클릭/전환 수치가 다를 수 있음** (어트리뷰션 방식 차이) — 항상 출처 명시
- 네이버는 자체 채널이므로 외부 채널(구글/메타)과 **ROAS를 단순 비교하지 말 것**
- 신규 소재 추가 시 네이밍 컨벤션 준수 필수
