## TAB 1 · Daily Check

### 예산 소진 섹션

**예산 파일 스펙**
- 경로: `data/raw/budget/YYYY-MM-DD_budget.csv`
- 컬럼: `채널(구글/메타/네이버)`, `일예산(원)`
- 없으면 해당 섹션 비활성화 + 안내 메시지 표시

**표시 방식**
- 채널별 게이지 차트 (0~100%)
- 소진율 구간별 색상:
  - 0~59%: 초록 (정상)
  - 60~79%: 노랑 (주의)
  - 80~100%: 빨강 (위험)
- 게이지 아래 `실제 비용 ₩X / 일예산 ₩Y (Z%)` 텍스트

### 이상치 플래그 섹션

아래 조건을 자동 감지해서 플래그 테이블 출력.

| 플래그 | 조건 | 심각도 |
|--------|------|--------|
| ROAS 미회수 | ROAS < 100% | 🔴 즉시 점검 |
| CPA 이상 | CPA > 채널 평균 × 3 | 🟡 주의 |
| 비용 급변 | 전일 대비 비용 ±30% | 🟡 주의 |

- 플래그 없으면 `✅ 이상 없음` 표시
- 플래그 테이블 컬럼: 심각도, 채널, 캠페인, 소재, 지표값, 기준값

### 전일 비용 증감 섹션

- 채널별 카드: 전일 비용 / 전전일 비용 / delta(%)
- delta 양수면 빨강 화살표, 음수면 초록 화살표
- 데이터가 1일치만 있으면 delta 미표시 (전전일 없음 안내)

## TAB 2 · 성과 Overview

기존 Overview + 채널 분석 탭 통합, 캠페인 요약 추가.

**구성 (위→아래)**
1. KPI 카드 8개: 노출, 클릭, CTR, 비용, 매출, ROAS(%), 구매, CPA
2. 일별 매출/비용 Bar + ROAS% 추이 Line (좌우 2열)
3. 채널 × 캠페인목적 ROAS 히트맵
4. 채널별 ROAS Bar / 비용 비중 Pie (좌우 2열)
5. 캠페인별 ROAS 랭킹 Bar

---

## TAB 3 · 소재 분석

기존 소재 탭 유지 + 메시지 카테고리 분석 추가.

**구성**
1. 소재 포맷별(VID/IMG/CRS/TXT) CTR 비교
2. 소재 포맷 × 채널 ROAS 히트맵
3. 메시지 카테고리별(플러스멤버십/적립혜택/할인쿠폰 등) ROAS 비교
4. 소재별 상세 성과 테이블
